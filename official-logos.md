# Color specifications
## Color Gold

**Pantone Yellow**  
HEX: #FFCC00  
RGB: 255, 204, 0  
CMYK: 0, 21, 100, 0  

## Color Azure

**Pantone Reflex Blue**   
HEX: #000099  
RGB: 0, 51, 153    
CMYK: 100, 80, 0, 0
